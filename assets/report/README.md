# Report Assets

This folder contains report-ready diagrams for the ICTU AI Chatbot project.

Files:

- `ai-agent-cover-figure.svg`: presentation or chapter opener visual
- `ai-agent-overview.svg`: high-level system architecture
- `rag-pipeline-flow.svg`: end-to-end AI agent workflow
- `hybrid-retrieval-architecture.svg`: retrieval strategy and evidence fusion

Suggested usage:

- Use `ai-agent-cover-figure.svg` on the title slide, chapter opener, or overview page.
- Use `ai-agent-overview.svg` in the architecture chapter.
- Use `rag-pipeline-flow.svg` in the AI agent or RAG workflow chapter.
- Use `hybrid-retrieval-architecture.svg` in the retrieval design or evaluation chapter.
